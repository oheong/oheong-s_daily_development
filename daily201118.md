```mysql
select deptno, sum(sal)
from emp
group by deptno
having sum(sal) > (select avg(sum(sal))
                    from emp
                    group by deptno);
```

= (같은 결과값을 내는 쿼리임)

```mysql
with dept_sum as (select deptno, sum(sal)as sum_sal
                    from emp
                    group by deptno)
select *
from dept_sum
where sum_sal > (select avg(sum_sal)
                    from dept_sum);
```

=> with문의 쿼리를 가장 먼저 한번❗ 실행하고 임시 테이블에 저장함. 이 임시테이블을 조회하여 사용
